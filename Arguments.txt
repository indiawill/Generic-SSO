GenericSSO :: Enterprise Single Sign-On Module

* Parameters will be always starting from Application Name configured in data.ini file as [SECTION]

For using parameter calls inside ini file use below commands in arguments section for APP option: 
%ip = First Argument in the parameter call after [SECTION] 
%hostname = Second Argument in the parameter call after [SECTION] 
%username = Third Argument in the parameter call after [SECTION] 
%password = Fourth Argument in the parameter call after [SECTION] 
%port = Fifth Argument in the parameter call after [SECTION] 

For using parameter calls for WEB option in URL section:
%ip = First Argument in the parameter call after [SECTION] 
%hostname = Second Argument in the parameter call after [SECTION] 

NOTE:: While using inFile parameter calls, it is mandatory to use arguments according to the above commands and with respective placeholder in parameter
